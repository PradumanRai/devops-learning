# Linux CPU Alert Monitoring Script (Gmail SMTP + Postfix)

## 📌 Project Overview
This project monitors CPU usage on Ubuntu/Linux and sends an email alert when CPU usage exceeds a defined threshold (90%).

The alert system uses:
- Bash Script
- Postfix Mail Server
- Gmail SMTP Relay (App Password Authentication)
- Cron Scheduler

---

## ⚙️ Environment Details
- OS: Ubuntu / Lubuntu 24.04
- Shell: Bash
- Mail Server: Postfix
- SMTP Relay: Gmail (smtp.gmail.com:587)
